(Tutorial Overview)

The real-time scheduling module uses a Quadratic Programming approach to determine priorities for event-triggered systems such that all deadlines are satisfied. The approach is based on this article, but it is not a complete re-implementation as it per default allows allows global cycles. Per default, the approach uses the Gurobi solver which has to be installed and the gurobi.jar has the be on the classpath.
