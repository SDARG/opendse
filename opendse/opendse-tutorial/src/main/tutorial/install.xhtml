Installation of OpenDSE

Version: @version@ - Date: @date@

(Tutorial Overview)

Gradle

OpenDSE relies on the build automation tool Gradle. Download Gradle from the website to use OpenDSE.

To start OpenDSE, execute the following command in the opendse directory:

opendse> gradle run

Gradle will download all neccessary dependencies from Maven. You can see additional build tasks with:

opendse> gradle tasks

Eclipse Integration

To create Eclipse .project and .classpath files execute:

opendse> gradle eclipse

Then, you can import the subprojects in Eclipse.
